Lipid Profile
Overview

A Lipid Profile measures fats (lipids) in the blood and is commonly used to assess cardiovascular risk.

Total Cholesterol
Description

Total amount of cholesterol present in blood.

Unit

mg/dL

Reference Range
Category	Value
Desirable	< 200
Borderline High	200–239
High	≥ 240
Triglycerides (TG)
Description

Primary storage form of fat in the body.

Unit

mg/dL

Reference Range
Category	Value
Normal	< 150
Borderline High	150–199
High	200–499
Very High	≥ 500
HDL Cholesterol (High-Density Lipoprotein)
Description

High-density lipoprotein cholesterol.

Unit

mg/dL

Reference Range
Men
Category	Value
Low	< 40
Acceptable	≥ 40
Protective	≥ 60
Women
Category	Value
Low	< 50
Acceptable	≥ 50
Protective	≥ 60
LDL Cholesterol (Low-Density Lipoprotein)
Description

Low-density lipoprotein cholesterol.

Unit

mg/dL

Reference Range
Category	Value
Optimal	< 100
Near Optimal	100–129
Borderline High	130–159
High	160–189
Very High	≥ 190
VLDL Cholesterol (Very Low-Density Lipoprotein)
Description

Very low-density lipoprotein cholesterol.

Unit

mg/dL

Reference Range
Category	Value
Normal	5–40
Common Calculation

VLDL = Triglycerides ÷ 5

(Valid when TG < 400 mg/dL)

Non-HDL Cholesterol
Description

Total cholesterol minus HDL cholesterol.

Formula

Non-HDL Cholesterol = Total Cholesterol − HDL Cholesterol

Reference Range
Category	Value
Optimal	< 130
Borderline High	130–159
High	160–189
Very High	≥ 190
Total Cholesterol / HDL Ratio
Formula

Total Cholesterol ÷ HDL

Reference Range
Category	Ratio
Ideal	< 3.5
Acceptable	3.5–5.0
Elevated	> 5.0
LDL / HDL Ratio
Formula

LDL ÷ HDL

Reference Range
Category	Ratio
Ideal	< 2.5
Acceptable	2.5–3.5
High	> 3.5
Triglyceride / HDL Ratio
Formula

Triglycerides ÷ HDL

Reference Range
Category	Ratio
Ideal	< 2
Moderate	2–4
High	> 4
Friedewald LDL Formula

Used when LDL is not directly measured.

Formula

LDL = Total Cholesterol − HDL − (Triglycerides ÷ 5)

Conditions
Fasting sample preferred
Triglycerides < 400 mg/dL
