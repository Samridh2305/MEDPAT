Diabetes Profile
Overview

Diabetes-related laboratory tests evaluate blood glucose control and help diagnose, monitor, and assess the risk of diabetes mellitus.

Fasting Blood Sugar (FBS) / Fasting Plasma Glucose (FPG)
Description

Blood glucose level measured after at least 8 hours of fasting.

Unit

mg/dL

Reference Range
Category	Value
Normal	70–99
Prediabetes	100–125
Diabetes	≥126
Random Blood Sugar (RBS) / Random Plasma Glucose
Description

Blood glucose level measured at any time regardless of meals.

Unit

mg/dL

Reference Range
Category	Value
Normal	<140
Prediabetes	140–199
Diabetes	≥200
Postprandial Blood Sugar (PPBS) / 2-Hour Post Meal Glucose
Description

Blood glucose level measured 2 hours after a meal.

Unit

mg/dL

Reference Range
Category	Value
Normal	<140
Prediabetes	140–199
Diabetes	≥200
Oral Glucose Tolerance Test (OGTT)
Description

Blood glucose measured 2 hours after consuming a standardized glucose solution.

Unit

mg/dL

Reference Range
Category	Value
Normal	<140
Prediabetes	140–199
Diabetes	≥200
HbA1c (Glycated Hemoglobin)
Description

Percentage of hemoglobin molecules bound to glucose.

Reflects average blood glucose over the previous 2–3 months.

Unit

%

Reference Range
Category	Value
Normal	<5.7
Prediabetes	5.7–6.4
Diabetes	≥6.5
Estimated Average Glucose (eAG)
Description

Estimated average blood glucose derived from HbA1c.

Unit

mg/dL

Formula

eAG = (28.7 × HbA1c) − 46.7

Example Values
HbA1c (%)	eAG (mg/dL)
5.0	97
6.0	126
7.0	154
8.0	183
9.0	212
10.0	240
Fasting Insulin
Description

Amount of insulin circulating in blood after fasting.

Unit

µIU/mL

Reference Range
Category	Value
Normal	2–25
C-Peptide
Description

Protein released when insulin is produced by the pancreas.

Unit

ng/mL

Reference Range
Category	Value
Normal	0.8–3.5
HOMA-IR (Homeostatic Model Assessment of Insulin Resistance)
Description

Calculated marker used to estimate insulin resistance.

Formula

When glucose is in mg/dL:

HOMA-IR = (Fasting Insulin × Fasting Glucose) ÷ 405

Reference Range
Category	Value
Normal	<2.0
Borderline	2.0–2.9
Insulin Resistance	≥3.0
Fructosamine
Description

Reflects average blood glucose over the previous 2–3 weeks.

Unit

µmol/L

Reference Range
Category	Value
Normal	200–285
Urine Glucose
Description

Glucose detected in urine.

Reference Range
Category	Result
Normal	Negative
Urine Ketones
Description

Ketone bodies detected in urine.

Reference Range
Category	Result
Normal	Negative
Urine Microalbumin
Description

Small amount of albumin present in urine.

Unit

mg/day

Reference Range
Category	Value
Normal	<30
Microalbuminuria	30–300
Macroalbuminuria	>300
Urine Albumin-to-Creatinine Ratio (UACR)
Description

Ratio used to detect early kidney involvement.

Unit

mg/g

Reference Range
Category	Value
Normal	<30
Moderately Increased	30–300
Severely Increased	>300
