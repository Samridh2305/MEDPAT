Thyroid Profile Knowledge Base
Overview

A Thyroid Profile evaluates the function of the thyroid gland, which regulates metabolism, growth, development, temperature regulation, heart rate, and energy utilization.

The thyroid gland produces:

Thyroxine (T4)
Triiodothyronine (T3)

Their production is controlled by:

Thyroid Stimulating Hormone (TSH) from the pituitary gland
Primary Thyroid Tests
Thyroid Stimulating Hormone (TSH)
Description

Hormone secreted by the pituitary gland that stimulates the thyroid gland to produce T3 and T4.

Unit

µIU/mL (mIU/L)

Typical Reference Range
Category	Value
Normal	0.4–4.5
Mildly Elevated	4.5–10
Significantly Elevated	>10
Low	<0.4
Biological Function
Regulates thyroid hormone production
Most sensitive screening marker for thyroid dysfunction
Alternative Names
TSH
Thyrotropin
