# MEDPAT Frontend

Frontend application for MEDPAT, a medical report interpretation and disease risk prediction platform.

## Overview

The MEDPAT frontend is built using React and Vite. It provides a user-friendly interface for:

* Uploading medical reports (PDFs)
* Viewing extracted laboratory values
* Asking questions about uploaded reports using AI-powered analysis
* Predicting disease risks based on extracted lab results

The frontend communicates with the FastAPI backend through REST APIs.

---

## Features

### Report Upload

* Upload PDF medical reports
* Receive extracted laboratory values from the backend
* Display report metadata and extracted results

### AI Question Answering

* Ask questions about uploaded reports
* Receive natural language explanations generated by the RAG pipeline

### Disease Risk Prediction

* Generate disease risk predictions based on extracted lab values
* Display diabetes, kidney disease, and hypertension risk percentages

### Responsive UI

* Modern card-based interface
* Loading states for uploads, questions, and predictions
* Clean presentation of laboratory results

## MEDPAT Workflow

1. Upload a medical report.
2. Backend extracts report text and laboratory values.
3. Extracted results are displayed in the UI.
4. Users can ask questions about their report.
5. AI generates explanations using Retrieval-Augmented Generation (RAG).
6. Disease prediction models estimate health risks based on extracted values.
